What is Siemens NX?

NX is an advanced high-end CAD/ CAM/ CAE. It is used for design, analysis and manufacturing. It is intended as a one stop shop (OSS) solution for design, manufacturing and team work in various engineering fields.

Data Management

Create a New Part

Use one of the following 4 methods to begin creating a new part:

  1. Go to Home Tab Ribbon Bar (Standard Group) New OR
  2. File New (Drop down menu top left of screen) OR
  3. Menu File New OR
  4. New (Quick access toolbar)

This will open a new session, asking for the part type, name, and location of the new file to be created. NX uses template files that are preconfigured for the purpose of the new part, i.e. modelling, drawing, assembly, sheet metal etc.

Such things as part units, display preference, toolbars, colours, layer setup, construction geometry and coordinate systems are saved in the template part files and forms, and serve to preconfigure the new part.

Existing template files can be customised to suit site requirements or new ones added. But for now, all that is required is to use the standard templates, specifying a name for the new file, a location to save, units and the type of part required.

Open an Existing Part

Use one of the following 4 methods to open an existing part:

  1. Go to Home Tab Ribbon Bar (Standard Group) Open OR
  2. File Open (Drop down menu top left of screen) OR
  3. Menu File Open OR
  4. Open (Quick access toolbar) OR
  5. History Palette

Save a Part

NX doesn’t automatically save part file, so it is imperative that you save your work frequently. If for some reasons, NX shuts down all work in the current session will be lost, the saved part files are protected from data corruption due to system crash but work between saves is not protected.

There are 5 save options:

  1. Save: This option will save the work part (on screen) and any modified components.
  2. Save Work Part Only: This option will only save the active part on the screen.
  3. Save As: This option allows you to save the part on screen using a different name and/or type. The default type is .prt. However, you can save your file as:
    • IGES (.igs)
    • STEP (.stp)
    • AutoCAD DXF (.dxf)
    • AutoCAD DWG (.dwg)
    • CATIA Model (.model)
    • CATIA V5 (.catpart)
    • ACIS text (.sat)
    • ACIS Binary (.sab)
    • STEP242 compressed (.stpz)
    • STEP242 XML (stpx)
    • STEP242 compressed XLM (.stpxz)
    • Wavefront ASCII (stpxz)
    • IFC (.ifc)
  4. Save All: This option will save all the opened and modified part files.
  5. Save Bookmark: This option will save a screenshot and context of the present model on the screen as a .JPEG file and bookmarks.

Closing and Exiting

Various options are available for closing parts with or without saving. On file exit NX checks if any parts have been modified in session but not saved and alerts user.

Absolute Coordinate System

Absolute Coordinate System: All objects are referenced from the absolute coordinate system. This is a fixed coordinate system and the locations and orientations of every object in NX modelling space are related back to this system.

Video - Siemens NX Curriculum - Absolute Coordinate System (3:10)

Work Coordinate System

The Work Coordinate System is what you will use for construction when you want to determine orientations and angles of features. The axes of the WCS are denoted:

  • XC
  • YC
  • ZC

The C stands for current. It is possible to have multiple coordinate systems in a part file, but only one of them can be the work coordinate system.

Layers

Layers are used to store objects in a file, and work like containers to collect the objects in a structured and consistent manner. Unlike simple visual tools like Show and Hide, Layers provide a permanent way to organize and manage the visibility and select-ability of objects in your file. A Layer is a system-defined attribute such as colour, font, and width that all objects in NX must have. There are 256 usable layers in NX, one of which is always the Work Layer. Any of the 256 layers can be assigned to one of four classifications of status.

  • Work
  • Selectable
  • Visible Only
  • Invisible

Move The Work Coordinate System

To move a component in relation to an existing Work Coordinate System, you can follow these steps:

  1. Hover the cursor over the component to reposition, then click MB3 Move. The Dynamic WCS will display in the work area and the Move Component dialog will display.
  2. On the Move Component dialog, set Motion = Dynamic.
  3. In the work area hover the cursor over the cone shape at the end of the X, Y, or Z axis on the Dynamic WCS.

Sketching

Introduction

In this chapter, you will learn how to create and edit sketches in NX. Typically, Modelling starts from a 2D sketch which is then Extruded, Revolved or Swept to create solids. Complex shapes that are otherwise very difficult to model can in most cases be broken down into a combination [join, remove, intersect] of other smaller features which are based on 2D profiles. This chapter provides an overview of sketching concepts and practice exercises.

Overview of Sketch functionality

NX sketch is a feature with named set of curves sketched on a single plane joined together as a string that when swept form a solid (or surface) geometry. When sketching the 2d curves it is the design intent that is being captured, not the exact dimensions and geometric constraints. These are added after.

After sketching is completed, dimensioned, and constrained it can be used to create a 3D shape. As an example, the sketch curves [i.e., the cross-section of the 3D geometry at that point] can be swept along a straight path as an Extrude feature, or circular path as a Revolve feature, or even a 3D path as a Sweep feature.

NX is a history based parametric modeller so associativity is created between sketch and 3D feature, the sketch [parent feature] can be modified and changes reflected in associated features, for example an extrude. Selecting objects one by one

You can select objects by clicking them individually, one after another, in any order. You can select objects that are not connected to each other, and of different type, ie curve, surface, body etc. The following example shows how to select curves one by one in a sketch.

  1. Move the cursor to the object that you want to select, as the cursor moves over an object the colour changes to the preselection colour (magenta) and you can now click to select the object.

  2. Once selected the object colour changes to the selected object colour (orange).

  3. You can continue clicking curves to complete your selection.

Deselecting objects

  1. To deselect a single object, press Shift key and click on a selected object.
  2. To deselect a group of objects, press Shift key and MB1 and create selection box over the objects you want to deselect. Once they are highlighted, release the buttons and those objects will be deselected.
  3. To deselect all, do one of the following:
    • Click Deselect All on the Top Border bar
    • Press Esc key
    • Click in the background of the graphics window with MB1
